  2. MingsNetwork
MingsNetwork is a collection of communities and services by Ming and partners.

  •   MingsUtilities  - (this website)

  •   MingsUtilities Discord Server  - You can join the MingsUtilities Discord server to get support for any of the services offered by MingsUtilities. You can also get support for other general tech things. Just ask a question and a community member will help you out.

  •   MingsUtilities Lemmy Community  - You can join the MingsUtilities Lemmy community to get support for any of the services offered by MingsUtilities or to discuss anything related to MingsUtilities.

  •   @MingsUtilities  - You can follow the MingsUtilities Mastodon account to get updates on MingsUtilities services and get updates on other things within the MingsUtilities community.

  •   MingTheNoob/issues/issues  - You can use this repository to create an issue to report bugs, request features, and more.


  •   Sources DB  - Sources DB is an opensource, crowdsourced, database of sources and learning materials for various topics. This is a community project separate from MingsUtilities, however still partnered, endorsed, and supported by MingsUtilities.

  •   Sources DB Discord Server  - The Sources DB Discord server is a place for the community to chat, discuss academic topics, share resources, and more.


  •   MingsHangout  - MingsHangout is a community focused on spreading privacy, security, and freedom through the use of opensource services and crowdsupported networks.

  •   MingsHangout Discord Server  - The MingsHangout Discord server is a place for the community to chat, discuss privacy, security, and freedom, share resources, and more.
