Web Extensions to help you on the web

Our Web Extensions are 100% free and opensource.

QuickLinks is a web extension that allows you to quickly navigate to web pages through the use of commands.

SuperScript is a web extension that can apply userscripts and userstyles to web pages.

MingsSearx allows you to set a Searx instance of your choice as the default search engine.


Userscripts and Userstyles

Our Userscripts and Userstyles are 100% free and opensource.

MingsUtilities will soon have some install-ready userscripts and userstyles available via the corresponding pages linked above. In addition to our userscripts and userstyles, you will soon be able to view and contribute your own userscripts and userstyles via the corresponding GitHub repositories: Userscripts & Userstyles
